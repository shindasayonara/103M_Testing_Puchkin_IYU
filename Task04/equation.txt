Чек-лист тестирования функции решения квадратного уравнения ax^2 + bx + c = 0

1. Два различных действительных корня
a = 1, b = -3, c = 2
D = 1
Ожидаемый результат: (1, 2)
Описание: стандартный случай квадратного уравнения с двумя корнями.

2. Один действительный корень
a = 1, b = 2, c = 1
D = 0
Ожидаемый результат: (-1,)
Описание: дискриминант равен нулю.

3. Нет действительных корней
a = 1, b = 0, c = 4
D = -16
Ожидаемый результат: None
Описание: отрицательный дискриминант.

4. Коэффициент a отрицательный
a = -1, b = 3, c = -2
Ожидаемый результат: (1, 2)
Описание: проверка корректной работы при отрицательном a.

5. Линейное уравнение (a = 0)
a = 0, b = 2, c = -4
Ожидаемый результат: (2,)
Описание: уравнение превращается в bx + c = 0.

6. Все коэффициенты равны нулю
a = 0, b = 0, c = 0
Ожидаемый результат: бесконечно много решений
Описание: особый случай.

7. b = 0
a = 1, b = 0, c = -9
Ожидаемый результат: (-3, 3)
Описание: симметричные корни.

8. c = 0
a = 1, b = -5, c = 0
Ожидаемый результат: (0, 5)
Описание: один корень равен нулю.
